    Data Scope Note

    Definition

    A note explaining the coverage, specialized usage, and/or rules for assigning the Date value.

    Required?

    No

    Repeatable?

    Yes


    Guidelines #

    When used in legacy EAIC collections, this property contains printed information on the backs of postcards, including indicia, that is relevant to dating the postcard itself (not the image reproduced in that postcard).
